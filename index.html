Let's Play Crystal Collector!

Instructions

    At the start of each game you will be given a "target" number.
    There are four colored crystals down below, each assigned a random number value that is hidden until you click them.
    Each new game will reset both the target number you are trying to reach, as well as the values of each crystal.
    You win the game by matching your score to the target number.
    Good Luck!!!
